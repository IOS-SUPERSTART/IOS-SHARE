### 전수민: 데이트 코스 짜기 프로그램 
            1. 현재 위치, 플레이스 지정하여 각 지역별 맛집, 놀거리 등등 플레이스 표시 
            2. 시설정버, 이용료, 주소, 전화번호등 플레이스 정보제공(공공데이터 활용해도 좋고 직접 입력도 가능)
            3. 짜여진 코스를 보여줄 수 있고, 내가 장소를 지정하여 제공된 플레이스를 기반으로 코스를 짤 수 있음 
            4. 코스가 완료되면 이동거리 등 네이게이션 처럼 이동거리, 이동시간 보여줌
